# Sevkiyat Yönetim Sistemi

Bu proje, Next.js kullanarak geliştirilmiş basit bir sevkiyat yönetim sistemidir. Yönetici sevkiyat oluşturabilir, depo sorumlusu ürünleri kontrol edebilir.

## Özellikler

- **Yönetici Paneli**: Sevkiyat oluşturma, tarih, adres ve ürünler girme. Sevkiyatları takip etme.
- **Depo Sorumlusu Paneli**: Gelen sevkiyatları görme, ürünleri kontrol etme, sorun bildirme veya tamamlandı olarak işaretleme.

## Teknoloji

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- JSON dosyası ile veri saklama

## Kurulum

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

3. Tarayıcıda [http://localhost:3000](http://localhost:3000) adresine gidin.

## Kullanım

- Ana sayfadan Yönetici veya Depo Sorumlusu paneline gidin.
- Yönetici: Yeni sevkiyat oluşturun.
- Depo: Sevkiyatları kontrol edin ve işlem yapın.
